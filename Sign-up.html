Sign-Up Flow

UX research and design for a delivery App

Project Overview

The project aims to create a user-friendly sign-up flow for Squirrels' new mobile app.
The goal is to provide a delightful and efficient user experience that encourages new clients to join the service and supports Squirrels' mission of sustainable local farming.

Problem Statement:

The client aims to attract a large user base for the delivery app while ensuring that only addresses within the designated range can receive deliveries.

  • This is a delivery app with a specified delivery range.
  • During the sign-up process, we need to determine whether the given address can be delivered.
  • If the provided address cannot be delivered, the user will be redirected to the product page.

My Role:

UX Designer (Research | User Flow | Wireframe | Prototype | UI Design)

Discover

research and key findings

Competitive Audit

For the competitive audit, I have chosen three popular delivery apps: Uber Eats, Instacart, and HelloFresh. The specific area of focus will be the signup methods and flow of these apps. By analyzing and comparing how these apps handle the signup process, we can gain insights into their user onboarding strategies and identify strengths and areas for improvement.

Design

User Flow

Research suggests that a two-step signup flow is essential, involving registration followed by address verification. This approach helps streamline the onboarding process, ensuring accurate delivery address information and enhancing user experience.

Wireframe

I utilized a screen shot collage to expedite the wireframing process, enabling me to quickly visualize and organize my ideas. This approach allowed for a rapid turnaround and facilitated clear thought in translating concepts into a tangible wireframe.

UI

I created the UI by seamlessly integrating elements from both the iOS design guidelines and the brand's own unique design guidelines. This approach ensured consistency with the iOS platform while maintaining the brand's distinct visual identity.
